# cell_confluency
U-net based cell confluency calculator for bright field cell image.

明視野顕微鏡の細胞領域抽出用Pythonプログラム。

一般的なフィルター処理に比べ、照明ムラなどの環境の変化に対する頑強性が高い。

Result
Adam(lr=0.5e-4) 50epochs
